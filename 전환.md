## Phase 2 — MaxPool 클래스 (1시간)

  

### 목표

2x2 max pooling layer 구현.

  

### 작업x

```python

class MaxPool(Node):

    def __init__(self, pool_h, pool_w, stride):

        self.pool_h = pool_h

        self.pool_w = pool_w

        self.stride = stride

        self.x_shape = None

        self.arg_max = None

    def forward(self, x):

        N, C, H, W = x.shape

        out_h = (H - self.pool_h) // self.stride + 1

        out_w = (W - self.pool_w) // self.stride + 1

        # im2col 활용 (Phase 3에서 만들 함수)

        col = im2col(x, self.pool_h, self.pool_w, self.stride, 0)

        col = col.reshape(-1, self.pool_h * self.pool_w)

        self.arg_max = np.argmax(col, axis=1)

        out = np.max(col, axis=1)

        out = out.reshape(N, out_h, out_w, C).transpose(0, 3, 1, 2)

        self.x_shape = x.shape

        return out

    def backward(self, dout):

        dout = dout.transpose(0, 2, 3, 1)

        pool_size = self.pool_h * self.pool_w

        dmax = np.zeros((dout.size, pool_size))

        dmax[np.arange(self.arg_max.size), self.arg_max.flatten()] = dout.flatten()

        dmax = dmax.reshape(dout.shape + (pool_size,))

        dcol = dmax.reshape(dmax.shape[0] * dmax.shape[1] * dmax.shape[2], -1)

        dx = col2im(dcol, self.x_shape, self.pool_h, self.pool_w, self.stride, 0)

        return dx

```

  

> 이건 im2col/col2im에 의존하므로 Phase 3을 먼저 끝내야 진짜 동작합니다. Phase 2는 "구조만 미리 짜놓기" 단계로 보세요.

  

### 공부할 것

- Pooling의 의미: spatial downsampling, translation invariance, 파라미터 수 감소

- max의 미분: argmax 위치만 1, 나머지 0

## Phase 3 — im2col / col2im (3~5시간) ★ 진짜 산

  

### 목표

4D conv 입력을 2D 행렬로 변환하는 두 유틸 함수.

  

### 핵심 개념

Conv 연산 = "모든 패치를 한 줄로 펴서 필터와 행렬곱"

  

```

원본 (N, C, H, W) → im2col → (N * out_H * out_W, C * kH * kW)

                                각 행 = 한 conv 패치를 펴놓은 것

필터 (C_out, C, kH, kW) → reshape → (C_out, C * kH * kW)

행렬곱 후 reshape → (N, C_out, out_H, out_W)

```

  

### Output size 공식 (외워둘 것)

```

out_size = (in_size + 2 * padding - kernel_size) / stride + 1

```

  

### 작업 (참고용 구현)

```python

def im2col(x, kh, kw, stride=1, pad=0):

    N, C, H, W = x.shape

    out_h = (H + 2*pad - kh) // stride + 1

    out_w = (W + 2*pad - kw) // stride + 1

    img = np.pad(x, [(0,0), (0,0), (pad,pad), (pad,pad)], mode='constant')

    col = np.zeros((N, C, kh, kw, out_h, out_w))

    for y in range(kh):

        y_max = y + stride * out_h

        for x_idx in range(kw):

            x_max = x_idx + stride * out_w

            col[:, :, y, x_idx, :, :] = img[:, :, y:y_max:stride, x_idx:x_max:stride]

    col = col.transpose(0, 4, 5, 1, 2, 3).reshape(N * out_h * out_w, -1)

    return col

  

def col2im(col, input_shape, kh, kw, stride=1, pad=0):

    N, C, H, W = input_shape

    out_h = (H + 2*pad - kh) // stride + 1

    out_w = (W + 2*pad - kw) // stride + 1

    col = col.reshape(N, out_h, out_w, C, kh, kw).transpose(0, 3, 4, 5, 1, 2)

    img = np.zeros((N, C, H + 2*pad + stride - 1, W + 2*pad + stride - 1))

    for y in range(kh):

        y_max = y + stride * out_h

        for x_idx in range(kw):

            x_max = x_idx + stride * out_w

            img[:, :, y:y_max:stride, x_idx:x_max:stride] += col[:, :, y, x_idx, :, :]

    return img[:, :, pad:H+pad, pad:W+pad]

```


## Phase 4 — Conv2D forward (1~2시간)

  

### 목표

Conv layer의 forward.

  

### 작업

```python

class Conv2D(Node):

    def __init__(self, W, b, stride=1, pad=0):

        self.W = W            # (FN, C, FH, FW)

        self.b = b            # (FN,)

        self.stride = stride

        self.pad = pad

        # backward용 cache

        self.x = None

        self.col = None

        self.col_W = None

        self.dW = None

        self.db = None

    def forward(self, x):

        FN, C, FH, FW = self.W.shape

        N, C, H, W = x.shape

        out_h = (H + 2*self.pad - FH) // self.stride + 1

        out_w = (W + 2*self.pad - FW) // self.stride + 1

        col = im2col(x, FH, FW, self.stride, self.pad)   # (N*oh*ow, C*FH*FW)

        col_W = self.W.reshape(FN, -1).T                  # (C*FH*FW, FN)

        out = col @ col_W + self.b                        # (N*oh*ow, FN)

        out = out.reshape(N, out_h, out_w, FN).transpose(0, 3, 1, 2)

        self.x = x

        self.col = col

        self.col_W = col_W

        return out

```

  

### 검증


## Phase 5 — Conv2D backward (옵션 A/C 선택 시, 2~4시간)

  

### 핵심 직관

Conv backward도 im2col 덕분에 결국 행렬곱 두 번 + col2im 한 번입니다. Affine backward와 같은 패턴.

  

| 구하려는 값 | 공식 |

|---|---|

| `db` | dout을 batch, H, W로 sum → (FN,) |

| `dW` | `col.T @ dout_reshaped`, 필터 모양으로 reshape |

| `dx` | `dout_reshaped @ col_W.T → col2im으로 4D 복원` |

  

### 작업

```python

def backward(self, dout):

    FN, C, FH, FW = self.W.shape

    dout = dout.transpose(0, 2, 3, 1).reshape(-1, FN)

    self.db = dout.sum(axis=0)

    self.dW = self.col.T @ dout

    self.dW = self.dW.transpose(1, 0).reshape(FN, C, FH, FW)

    dcol = dout @ self.col_W.T

    dx = col2im(dcol, self.x.shape, FH, FW, self.stride, self.pad)

    return dx

```

  

### 검증 — Gradient Check (필수!!!)

  

수치 미분과 해석 미분 비교. **이거 없이 conv backward 디버깅하면 하루 날아갑니다.**

  

```python

def numerical_gradient(f, x, eps=1e-4):

    grad = np.zeros_like(x)

    it = np.nditer(x, flags=['multi_index'])

    while not it.finished:

        idx = it.multi_index

        orig = x[idx]

        x[idx] = orig + eps; fp = f()

        x[idx] = orig - eps; fm = f()

        grad[idx] = (fp - fm) / (2 * eps)

        x[idx] = orig

        it.iternext()

    return grad

  

# 사용 예

x_small = np.random.randn(1, 1, 5, 5)

W_small = np.random.randn(2, 1, 3, 3) * 0.1

b_small = np.zeros(2)

t_small = np.array([[1, 0]])

  

conv = Conv2D(W_small, b_small)

loss_layer = SoftmaxWithLoss()

flat = Flatten()

aff = Affine(np.random.randn(2*3*3, 2)*0.1, np.zeros(2))

  

def loss_fn():

    out = conv.forward(x_small)

    out = flat.forward(out)

    out = aff.forward(out)

    return loss_layer.forward(out, t_small)

  

# 해석 미분

loss_fn()

dout = loss_layer.backward()

dout = aff.backward(dout)

dout = flat.backward(dout)

dout = conv.backward(dout)

  

# 수치 미분

num_grad_W = numerical_gradient(loss_fn, conv.W)

  

# 비교 — 1e-7 ~ 1e-5 정도면 OK

print(np.abs(conv.dW - num_grad_W).max())

```

## Phase 6 — CNN 모델 조립 (1시간)

  

### 목표 구조

```python

입력 (N, 1, 28, 28)

  ↓ Conv(1→16, 3x3, pad=1) → ReLU → MaxPool(2,2)   → (N, 16, 14, 14)

  ↓ Conv(16→32, 3x3, pad=1) → ReLU → MaxPool(2,2)  → (N, 32, 7, 7)

  ↓ Flatten                                          → (N, 1568)

  ↓ Affine(1568→64) → ReLU                          → (N, 64)

  ↓ Affine(64→10) → SoftmaxWithLoss                 → (N, 10)

```

  

### 작업

```python

def init_network(self):

    # He 초기화 (conv용)

    W1 = np.random.randn(16, 1, 3, 3) * np.sqrt(2 / (1*3*3))

    b1 = np.zeros(16)

    W2 = np.random.randn(32, 16, 3, 3) * np.sqrt(2 / (16*3*3))

    b2 = np.zeros(32)

    W3 = np.random.randn(32*7*7, 64) * np.sqrt(2 / (32*7*7))

    b3 = np.zeros(64)

    W4 = np.random.randn(64, 10) * np.sqrt(2 / 64)

    b4 = np.zeros(10)

    self.layers['Conv1']    = ly.Conv2D(W1, b1, stride=1, pad=1)

    self.layers['ReLU1']    = ly.ReLU()

    self.layers['Pool1']    = ly.MaxPool(2, 2, stride=2)

    self.layers['Conv2']    = ly.Conv2D(W2, b2, stride=1, pad=1)

    self.layers['ReLU2']    = ly.ReLU()

    self.layers['Pool2']    = ly.MaxPool(2, 2, stride=2)

    self.layers['Flatten']  = ly.Flatten()

    self.layers['Affine1']  = ly.Affine(W3, b3)

    self.layers['ReLU3']    = ly.ReLU()

    self.layers['Affine2']  = ly.Affine(W4, b4)

    self.last_layer         = ly.SoftmaxWithLoss()

```
