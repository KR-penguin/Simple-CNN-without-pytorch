🧮 합성곱(Convolution) 전체 과정 시뮬레이션

im2col.html은 합성곱의 1단계(이미지 → col 행렬)만 보여줍니다.
실제 합성곱은 거기서 필터를 col_W로 펴고 → 행렬곱(내적) → 다시 4D로 reshape 하는 전 과정입니다. 그 전부를 한 화면에서 따라가 봅니다.
input(N,C,H,W) ─im2col→ col  @  W(FN,C,FH,FW) ─reshape→ col_W  +  b  =  out(2D) ─reshape→ feature map(N,FN,OH,OW)
규격을 정한 뒤 ▶ 또는 ⏭ 버튼을 누르세요.
내적(dot product) 계산식이 여기에 단계별로 표시됩니다.
Convolution forward 코드
def forward(self, x):
FN, C, FH, FW = self.W.shape
N, C, H, W = x.shape
out_h = (H + 2*pad - FH)//stride + 1
out_w = (W + 2*pad - FW)//stride + 1
col = im2col(x, FH, FW, stride, pad)
col_W = self.W.reshape(FN, -1).T
out = col @ col_W + self.b
out = out.reshape(N, out_h, out_w, FN)
.transpose(0, 3, 1, 2)
return out
# col : (N·OH·OW, C·FH·FW)
# col_W : (C·FH·FW, FN)
# out : (N·OH·OW, FN)
